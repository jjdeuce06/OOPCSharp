# OOP Assignments Rewritten in C#

This repository contains a collection of assignments originally completed in my Object-Oriented Programming course using C++. I am currently rewriting these projects in C# as a way to strengthen my understanding of the language and improve my object-oriented design skills across multiple programming environments.

## Purpose

The goal of this repository is to:

- Practice C# syntax and conventions
- Reinforce object-oriented programming concepts
- Compare implementation differences between C++ and C#
- Gain experience with the .NET ecosystem and Visual Studio
- Improve problem-solving and software design skills through translation and refactoring

## Topics Covered

These rewritten assignments may include concepts such as:

- Classes and objects
- Encapsulation
- Constructors and destructors
- Inheritance and polymorphism
- File I/O
- Arrays and collections
- Exception handling
- Data validation
- Console-based applications

## Technologies Used

- C#
- .NET
- Visual Studio / VS Code

## Notes

The original assignments were developed in C++ as part of coursework for my Computer Science degree. The C# versions are personal rewrites intended for learning, practice, and professional development.

As I continue learning the language, projects in this repository may be refactored or expanded with more advanced C# features and best practices.
